# 새벽, 숲 — 손끝으로 깨우는 아침

**▶ 바로 체험하기: https://lgw7126.github.io/Claude-interactive_02/**

카메라 앞에서 손을 펼치면 안개 낀 밤의 숲에 아침이 옵니다.
영상 파일 없이 WebGL 셰이더가 매 프레임 안개·능선·여명을 실시간으로 그려내는 인터랙티브 랜딩페이지입니다.

## 인터랙션

| 동작 | 효과 |
|------|------|
| ✊ 주먹을 쥐면 | 별이 뜬 밤 — 안개가 짙게 피어오릅니다 |
| 🖐 손을 펼치면 | 여명 — 골짜기 사이로 태양빛이 스며들고 안개가 금빛으로 걷힙니다 |
| 👋 손을 움직이면 | 손이 있는 자리의 안개가 갈라집니다 |

카메라가 없거나 권한을 거부하면 마우스로 체험할 수 있습니다.
(마우스 이동 = 손 위치, 마우스 버튼 누르고 있기 = 손 펼침)

## 실행 방법

### 1. 온라인 (권장)

위 링크를 열고 카메라 권한을 **허용**하면 끝입니다.
크롬 · 엣지 · 사파리 최신 버전을 권장합니다.

### 2. 내 컴퓨터에서 실행

카메라 권한은 보안 정책상 `https` 또는 `localhost`에서만 동작하므로,
`index.html`을 더블클릭으로 여는 대신 로컬 서버로 열어야 합니다.

```bash
# 저장소 받기
git clone https://github.com/lgw7126/Claude-interactive_02.git
cd Claude-interactive_02

# 로컬 서버 실행 (파이썬 내장 서버)
python -m http.server 8000
```

브라우저에서 http://localhost:8000 접속 → 카메라 권한 허용.

> Node.js가 있다면 `npx serve` 로도 실행할 수 있습니다.

## 만든 방식

- **손 인식** — [Google MediaPipe Hand Landmarker](https://ai.google.dev/edge/mediapipe/solutions/vision/hand_landmarker)가 브라우저에서 손 관절 21개를 추적합니다. 손가락 끝~손목 거리로 펼침 정도(0~1)를, 손바닥 중심으로 위치를 계산합니다.
- **풍경 렌더링** — WebGL 프래그먼트 셰이더가 fbm 노이즈로 능선 5겹, 도메인 워핑 안개, 상승 안개 기둥, 여명 글로우를 매 프레임 생성합니다. 이미지·영상 파일이 전혀 없습니다.
- **파일 구성** — `index.html` 단 하나. 외부 의존성은 MediaPipe(CDN)와 Noto Serif KR 웹폰트뿐입니다.
